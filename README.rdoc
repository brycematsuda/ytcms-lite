== ytcms-lite

A simple CMS for managing content on YouTube using sqlite3.

== Steps to deploy
<code>
git clone https://github.com/brycematsuda/ytcms-lite.git<br>
cd ytcms-lite<br>
bundle install --without production<br>
bundle exec rake db:migrate<br>
bundle exec rake db:create db:migrate<br>
bundle exec rake db:test:prepare<br>
rails server<br>
</code>

And go to http://localhost:3000 in your web browser.
